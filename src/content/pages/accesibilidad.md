Creemos firmemente que Internet debe estar disponible y accesible para todos, y estamos comprometidos a
proporcionar un sitio web accesible para la audiencia más amplia posible, independientemente de las circunstancias
y la capacidad.

Nuestro objetivo es adherirnos a las Directrices de Accesibilidad al Contenido Web 2.1 (WCAG 2.1) del World Wide
Web Consortium (W3C) en el nivel AA.

Si has encontrado un problema de accesibilidad o tienes ideas para mejorar, puedes contactarnos a través de
nuestro [formulario de contacto](/contacto/) o llamando al [910 52 74 99](tel:+34910527499).

El sitio web está diseñado adaptándose a los estándares vigentes de accesibilidad, optimizado para las últimas
versiones de Chrome, Edge, Firefox, Safari y Opera, y con diseño responsive para cualquier dispositivo.
