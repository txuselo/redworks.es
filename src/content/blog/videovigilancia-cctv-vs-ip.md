Elegir entre **cámaras CCTV analógicas** y **cámaras IP** es una de las primeras decisiones al diseñar un sistema de videovigilancia para una empresa. Las analógicas transmiten la señal de vídeo por cable coaxial hasta un grabador dedicado (DVR); las IP digitalizan el vídeo y lo envían por red (Ethernet o wifi) hasta un grabador de red (NVR) o la nube. La elección correcta depende del tamaño de las instalaciones, la calidad de imagen necesaria y si ya existe cableado de red aprovechable.

## Diferencias principales

| | CCTV analógico | CCTV IP |
|---|---|---|
| Cableado | Coaxial, punto a punto hasta el DVR | Ethernet (PoE), aprovecha la red de datos |
| Calidad de imagen | Buena, limitada a resoluciones estándar | Alta resolución, 2 a 8 megapíxeles o más |
| Coste inicial | Más bajo en instalaciones pequeñas | Más alto por cámara, pero escala mejor |
| Acceso remoto | Depende del DVR, más limitado | Nativo, pensado para la nube y el móvil |
| Integración | Limitada a videovigilancia | Se integra con control de accesos y otros sistemas |

## Cuándo tiene sentido el CCTV analógico

El CCTV analógico sigue siendo una opción razonable cuando:

- La instalación ya dispone de cableado coaxial funcionando y no requiere ampliarse.
- El presupuesto es ajustado y la prioridad es cubrir puntos básicos sin necesidad de alta resolución.
- No es necesario el acceso remoto avanzado ni la integración con otros sistemas de seguridad.

## Cuándo conviene pasar a CCTV IP

El CCTV IP es la opción más adecuada cuando:

- Se necesita identificar caras, matrículas o detalles finos en la imagen.
- Se quiere consultar las cámaras en remoto desde el móvil de forma nativa y segura.
- Las instalaciones son grandes o están distribuidas en varios edificios conectados por red.
- Se planea integrar la videovigilancia con control de accesos u otros sistemas de seguridad.
